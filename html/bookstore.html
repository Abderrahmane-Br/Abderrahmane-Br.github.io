Bookstore

Bookstore is a front-end app that simulates e-commerce book selling service using google books api.

This project is my attempt to design and build an app large and complex enough to challenge my skills and put everything I know about front-end development in one project, covering everything from a modern trendy design, high responsivity, routing, caching, complex state management to rest api consuming and version controlling.

Live Github

Features

  • Glassmorphic UI
  • High responsive UI: screen range goes from 280px to 4k+
  • Loading skeletons
  • Caching api calls and crucial parts of the app state
  • Live search
  • Dynamic custom scrollbars

Stack and tools

ReactJS Redux React router
Javascript Scss Html5/Css3
Git/Github Figma Adobe XD
